Introduction

随着汽车智能互联需求的增加，越来越多的汽车厂商逐步推出具有网联功能的汽车。在智能网联环境下，汽车所遭受的攻击面越来越广，所面临的安全威胁也愈为严重。本项目从车企的实际安全需求出发，分析智能网联汽车所面临的安全威胁，分别从车载互联网关、车内CAN网络这两个层次研究针对智能网联汽车的多层次入侵检测模型，并实现相应的入侵检测系统，以检测针对汽车的恶意攻击行为，从而提高智能网联汽车的安全性。

Duised sitDamet

Function 1：车载CAN网络入侵检测

针对目前应用最广泛的车载网络协议CAN网络，分析可能存在的安全威胁，从报文角度，综合利用多种方法制作入侵检测工具，检测方法包括Entropy-HTM、报文周期性、ID序列分析、CART决策树、卷积神经网络等。

Details

Function 2：车载互联网关入侵检测


分析车载互联网关可能存在的安全威胁，进行数据的采集、分析、检测，采集数据包括进程信息、网络连接、资源使用率、网络流量、系统时间、日志信息等。对正常行为进行建模并检测报警异常行为。

Details

高集成度

车载互联网关对外连接互联网络，对内连接车内网络，如CAN,LIN,FlexRay等，而车内CAN网络是目前使用最为广泛的车内通信协议，是一种串行数据通信协议，有较大的安全隐患。本项目 从车载互联网关和车内CAN网络两个角度对汽车进行入侵检测，分析方法完备。


Details

多层次

面向车内CAN网络设计实现了多层次的入侵检测系统，本项目首先设计了五个基本分类器，分别为转移矩阵、CART决策树、ANN、HTM、LSTM，这些基本分类器 除LSTM都是二分类算法，在此基础上通过集成算法WMA综合基本分类器，将基本分类器的结果经过集成运算提升入侵检测结果的准确性并在一定程度上实现多分类。

Details

用户友好

本项目基于B/S架构利用Django框架完成设计，前后端分离，复杂的数据处理和分析对用户透明，操作简便。数据的输出调用High Charts等制作图表来展现。 相关信息用户可以直观地看到，便于入侵警告。
